git status
-> Verifica as modificações atuais

git log
-> Mostra o histórico de commits

git checkout --<PATH>
-> Desfaz todas as modificações NÃO estejam no stage desde do último commit

git checkout HEAD --<PATH>
-> Desfaz todas as modificações que esteja no stage desde do último commit

git reset <commit>
-> Reverte o commit e adiciona um commit com essa reversão

git reset --hard <commit>
-> Reverte o commit e adiciona um commit com essa reversão
(CUIDADO!, se for enviado para um repo remoto não é recomendado)
